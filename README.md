# interMSA Bot

> A bot for the interMSA's discord


## 🎯 Features

- Add roles based on reaction

## ⚡ Contribute

### Setup environment

```bash
mkdir interMSA-bot
cd interMSA-bot
npm install
```

### Run Bot

```bash
node bot.js
```
